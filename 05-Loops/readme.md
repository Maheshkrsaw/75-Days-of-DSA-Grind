## Concepts Covered:
- **Loop Basics**: Repetition of tasks to reduce code redundancy
- **while Loop**: Executes as long as the condition is true  
    - Print numbers 1 to 10  
    - Print numbers 1 to N  
    - Sum of first N natural numbers
- **for Loop**: Ideal for known iterations  
    - Print square patterns  
    - Print reverse of a number  
    - Reverse any given number
- **do-while Loop**: Executes at least once, then checks condition
- **Break Statement**: Exit loop early  
- **Continue Statement**: Skip current iteration and move to next
- **Prime Number Check**: Using loops and conditionals
